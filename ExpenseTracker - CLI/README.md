# Expense Tracker - CLI
A simple Python-based command-line application to help users manage their daily finances. This tool allows you to add, update, delete, list, and summarize your expenses, storing all data in a JSON file.

## Features
* **Add Expenses**: Record new transactions with a date, category, and description.
* **View & Summarize:** Display a clean, formatted list of all saved expenses, complete with a grand total and an automatic mathematical breakdown by category.
* **Update Records:** Select a specific expense from your list and edit its details to fix mistakes or update pricing.
* **Delete Records:** Remove an expense entirely from your history.
* **Data Persistence:** Automatically saves and loads data using a local `expenses.json` file so you never lose your history.

# How to Run the App

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/eddiee-jnr/PythonProjects.git](https://github.com/eddiee-jnr/PythonProjects.git)
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd "ExpenseTracker - CLI"
    ```
3.  **Run the script:**
    ```bash
    python expenseTracker.py
    ```
    *(Note: Depending on your system, you may need to use `python3` instead of `python`)*
